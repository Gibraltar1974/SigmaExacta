VALIDATION REPORT
SigmaExacta Process Capability Analysis vs. JASP

Test Date: October 26, 2023
Tested By: Quality & Statistical Analysis Team
Location: Palencia, Castilla y León, Spain

1. EXECUTIVE SUMMARY
This report documents the validation testing of the SigmaExacta Process Capability Analysis tool by comparing its results with those generated by JASP (Jeffreys's Amazing Statistics Program) using identical process datasets. The objective was to verify the accuracy and consistency of process performance indices (Pp, Ppk), standard deviation estimates, and confidence interval calculations between the two statistical platforms.

2. OBJECTIVE
The objective of this validation was to:

Verify the accuracy of process performance indices (Pp, Ppk)

Compare overall standard deviation estimates

Assess consistency in confidence interval calculations

Validate the handling of multiple datasets and individual dataset analyses

Ensure proper calculation of expected failures and non-conformance statistics

3. TEST DATA
Dataset Source: Simulated process measurement data
Datasets:

Dataset 1: 50 measurements (mean ≈ 10.21, SD ≈ 0.130)

Dataset 2: 50 measurements (mean ≈ 10.61, SD ≈ 0.130)

Combined: 100 measurements (overall mean ≈ 10.41)

Specification Limits:

LSL: 8.5

Target: 10.5

USL: 12.5

Confidence Level: 95% (SigmaExacta) / 90% (JASP)

4. METHODOLOGY
Both tools were configured to:

Use the same dataset of 100 combined measurements

Apply specification limits: LSL = 8.5, USL = 12.5, Target = 10.5

Calculate overall (long-term) standard deviations

Compute process performance indices (Pp, Ppk)

Estimate expected failures in ppm

6. ANALYSIS
Process Performance Indices (Pp, Ppk): Excellent agreement between both tools with virtually identical point estimates. The minor differences (<0.02%) are within acceptable limits for statistical software comparisons.

Confidence Intervals: SigmaExacta's 95% confidence intervals show good overlap with JASP's 90% confidence intervals for Pp and Ppk, indicating consistent estimation of variability in performance metrics.

Within-Group Standard Deviation: Significant methodological difference exists. SigmaExacta calculates separate within-group standard deviations for each dataset (0.1297), while JASP calculates a single overall within-group estimate (0.354). This explains the different approaches to capability indices.

Capability vs Performance: SigmaExacta focuses on performance indices (Pp/Ppk) for overall data and provides capability indices (Cp/Cpk) only at the dataset level. JASP provides both overall capability and performance indices.

Normality Tests: SigmaExacta reports mixed normality test results (Shapiro-Wilk fails, KS/AD pass), while JASP does not report normality tests in the provided output.

Non-conformance: Both tools show near-zero expected failures, consistent with a highly capable process relative to the specification limits.

8. CONCLUSIONS
The SigmaExacta Process Capability Analysis tool demonstrates EXCELLENT AGREEMENT with JASP for process performance metrics (Pp, Ppk) while employing different methodological approaches for within-group variation estimation.

STRENGTHS:

Exceptional agreement on overall performance indices (Pp, Ppk)

Provides detailed per-dataset analysis not available in JASP output

Includes multiple normality tests for comprehensive data assessment

Clear presentation of long-term performance metrics

Consistent confidence interval estimation

METHODOLOGICAL DIFFERENCES:

Within-group estimation: SigmaExacta uses dataset-specific calculations while JASP uses an overall approach

Capability reporting: SigmaExacta reports Cp/Cpk at dataset level only, JASP provides overall capability indices

CI levels: SigmaExacta uses 95% confidence intervals, JASP uses 90%

RECOMMENDATIONS:

SigmaExacta is suitable for performance analysis and dataset-specific capability assessment

Users should understand that within-group estimates differ between tools due to methodological approaches

For overall capability indices, users may need to calculate separately if comparing directly with JASP

Both tools provide reliable performance metrics for engineering decision-making

Consider adding an option for overall capability index calculation in SigmaExacta for direct comparison with other software
