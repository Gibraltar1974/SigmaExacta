6. ANALYSIS
6.1 Process Performance Indices (Pp, Ppk)
The point estimates agree perfectly with theoretical calculations and with JASP. Differences are negligible (<0.1%).

6.2 Confidence Intervals
SigmaExacta’s confidence intervals for Pp and Ppk now follow the standard formulas (chi‑square for Pp, Bissell for Cpk/Ppk). As a result, the 95% interval is wider than JASP’s 90% interval, as expected. The earlier anomaly (where SigmaExacta’s 95% interval matched JASP’s 90%) has been resolved.

6.3 Expected Failures
Both tools report identical expected failure rates (118,641 ppm), confirming the accuracy of the normal‑based probability calculations.

6.4 Normality Assessment
Both tools correctly identify the combined data as non‑normal (Shapiro‑Wilk p < 0.05; Anderson‑Darling critical value exceeded). SigmaExacta provides full test statistics.

6.5 Within‑Group Variation
SigmaExacta now reports σ within = 0.3930 and explains that the pooled‑SD‑with‑c₄ method was automatically chosen because subgroups are of equal size >10 (c₄ = 0.9869 for n=20). JASP reports a different value (0.538), likely due to a different method (e.g., moving range). The methodological difference is now clearly documented, allowing users to interpret results appropriately.

6.6 Statistical Control Assumption (Important Methodological Note)
*Process capability analysis assumes that the process is in a state of statistical control (i.e., only common‑cause variation present). In this test dataset, the ratio σ_total / σ_within = 1.19 (>1) and the subgroup means differ by up to 0.62 units (≈14σ within‑group), indicating that the process is not stable. Therefore, the calculated Pp/Ppk indices are presented here solely to validate the tool’s calculations; for a real process, a control chart should first be used to verify stability before interpreting capability indices.*

SigmaExacta provides I‑charts alongside capability results to help users assess stability.
