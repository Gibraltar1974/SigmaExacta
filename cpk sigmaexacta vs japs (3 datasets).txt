This report documents the validation testing of the SigmaExacta Process Capability Analysis tool against JASP (Jeffreys’s Amazing Statistics Program) using a dataset with three distinct subgroups. 6. ANALYSIS
6.1 Process Performance Indices (Pp, Ppk)
The point estimates agree perfectly with theoretical calculations and with JASP. Differences are negligible (<0.1%).

6.2 Confidence Intervals
SigmaExacta’s confidence intervals for Pp and Ppk now follow the standard formulas (chi‑square for Pp, Bissell for Cpk/Ppk). As a result, the 95% interval is wider than JASP’s 90% interval, as expected. The earlier anomaly (where SigmaExacta’s 95% interval matched JASP’s 90%) has been resolved.

6.3 Expected Failures
Both tools report identical expected failure rates (118,641 ppm), confirming the accuracy of the normal‑based probability calculations.

6.4 Normality Assessment
Both tools correctly identify the combined data as non‑normal (Shapiro‑Wilk p < 0.05; Anderson‑Darling critical value exceeded). SigmaExacta provides full test statistics.

6.5 Within‑Group Variation
SigmaExacta now reports σ within = 0.3879 and explains that the pooled‑SD‑with‑c₄ method was automatically chosen because subgroups are of equal size >10. JASP reports a different value (0.538), likely due to a different method (e.g., moving range). The methodological difference is now clearly documented, allowing users to interpret results appropriately.
