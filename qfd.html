What is Quality Function Deployment?

Quality Function Deployment (QFD) translates customer requirements ("Voice of the Customer") into engineering specifications. The House of Quality matrix links what the customer wants to how the organization can deliver it.

1. Customer NeedsWhat do they want?
2. Technical SpecsHow can we achieve it?
3. RelationshipsHow do they connect?
4. PrioritiesWhat's most important?

How to Use This Tool

  • Navigate to Input Data to add customer needs and technical specs.
  • Click directly on any name to edit it inline. Use the green + / red - buttons.
  • Define Relationships and Correlations using the internal tabs.
  • The Results & Matrix tab shows the House of Quality automatically.
  • Use Load Example for a demo, or Export to save everything (revision history included).
