What is Weibull Analysis?

Weibull Analysis is a powerful statistical method used in reliability engineering to model time-to-failure data. It is highly versatile and can model the failure characteristics of complex systems, from infant mortality to wear-out failures.

The standard 2-parameter Weibull distribution is defined by:

  • Beta (β) — Shape Parameter: Describes the failure mode and how failure rate evolves.
    • β < 1: Decreasing failure rate (infant mortality — burn-in failures).
    • β = 1: Constant failure rate (random failures — exponential distribution).
    • β > 1: Increasing failure rate (wear-out or fatigue failures).
    • β ≈ 3.5: Approximates a normal distribution.
  • Eta (η) — Scale Parameter (Characteristic Life): The time at which 63.2% of the population is expected to have failed, regardless of β.

3-Parameter Weibull NEW IN THIS TOOL

The 3-parameter Weibull adds a third parameter:

  • Gamma (γ) — Location Parameter (Threshold / Minimum Life): The guaranteed failure-free period. No failures can occur before time γ. All standard Weibull formulas apply to the shifted time (t − γ).

Use 3-parameter Weibull when:

  • There is a physical reason no failures occur before a minimum time (e.g., a component tested and verified before deployment).
  • The 2-parameter probability plot shows distinct curvature (non-linear data).
  • You have at least 5–10 failure observations.

Why is Weibull Analysis Useful?

Originally developed by Waloddi Weibull, this analysis is a cornerstone of modern reliability engineering. It allows engineers and managers to make data-driven decisions about:

  • Maintenance Planning: Predict when parts are likely to fail and schedule proactive maintenance.
  • Warranty Analysis: Forecast warranty claims and associated costs.
  • Design Improvement: Identify failure modes to guide product improvements.
  • Risk Assessment: Quantify the reliability of a component over its life cycle.
  • Spare Parts Optimization: Stock the right number of spares at the right time.

How to Use This Tool

  1. Go to the Calculator tab.
  2. Choose 2P or 3P Weibull — toggle the "3-Parameter Weibull" option if you suspect a threshold life.
  3. Select Calculation Method — MLE is recommended for small samples and censored data; Linear Regression is the classic approach.
  4. Enter Failure Times — comma or newline separated.
  5. Enter Suspension Times (optional) — times of units that had NOT yet failed when the test ended.
  6. Enter Interval-Censored Data (optional) — pairs like "100-150, 200-250".
  7. Click "Analyze" in the top toolbar — results and plots appear on the Results tab automatically.
  8. Interpret: Check β for failure mode, η for characteristic life, γ (if 3P) for minimum life, MTTF for mean life. Review the probability plot for goodness-of-fit.
  9. Save to History & Export — click "Save this analysis to history" (inside the results panel) to accumulate multiple analyses, then "Save As" in the toolbar to open the export dialog and download all results to Excel.

History & Origin

The Weibull distribution was first described by Swedish engineer Waloddi Weibull in 1937, and formally published in his landmark 1951 paper "A Statistical Distribution Function of Wide Applicability" presented to ASME. Its flexibility to model virtually any failure pattern through the shape parameter β made it revolutionary and universally adopted in aerospace, automotive, manufacturing, electronics, and medical device industries.

Mathematical Formulas

2-Parameter Weibull (β, η)

PDF

$$f(t) = \frac{\beta}{\eta}\left(\frac{t}{\eta}\right)^{\beta-1} e^{-(t/\eta)^{\beta}}$$

CDF

$$F(t) = 1 - e^{-(t/\eta)^{\beta}}$$

Reliability Function

$$R(t) = e^{-(t/\eta)^{\beta}}$$

Hazard (Failure Rate)

$$h(t) = \frac{\beta}{\eta}\left(\frac{t}{\eta}\right)^{\beta-1}$$

MTTF

$$MTTF = \eta\,\Gamma\!\left(1 + \frac{1}{\beta}\right)$$

B-Life Percentile

$$t_p = \eta\left[-\ln(1-p)\right]^{1/\beta}$$
3-Parameter Weibull (β, η, γ) — Location Parameter γ

The 3-parameter Weibull shifts the distribution by γ. All formulas apply for t > γ. When γ = 0, the distribution reduces to the standard 2-parameter form.

PDF

$$f(t) = \frac{\beta}{\eta}\!\left(\frac{t-\gamma}{\eta}\right)^{\!\beta-1}\! e^{-\left(\frac{t-\gamma}{\eta}\right)^{\!\beta}},\quad t>\gamma$$

CDF

$$F(t) = 1 - e^{-\left(\frac{t-\gamma}{\eta}\right)^{\!\beta}},\quad t>\gamma$$

Reliability Function

$$R(t) = e^{-\left(\frac{t-\gamma}{\eta}\right)^{\!\beta}}$$

Hazard

$$h(t) = \frac{\beta}{\eta}\!\left(\frac{t-\gamma}{\eta}\right)^{\!\beta-1}$$

MTTF

$$MTTF = \gamma + \eta\,\Gamma\!\left(1 + \frac{1}{\beta}\right)$$

B-Life Percentile

$$t_p = \gamma + \eta\left[-\ln(1-p)\right]^{1/\beta}$$
MLE Log-Likelihood (with censoring)

Log-Likelihood

$$\ell(\beta,\eta,\gamma) = r\ln\beta - r\beta\ln\eta + (\beta-1)\sum_{i=1}^{r}\ln(t_i-\gamma) - \sum_{j=1}^{n}\!\left(\frac{t_j-\gamma}{\eta}\right)^{\!\beta}$$

AIC / BIC

$$AIC = 2k - 2\ell \qquad BIC = k\ln(n) - 2\ell$$

k = number of parameters (2 or 3), n = total observations
